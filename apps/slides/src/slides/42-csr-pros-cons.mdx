## CSR Pros & Cons

- ✅ Simple mental model — everything runs in the browser
- ✅ Cheap static hosting — any CDN can serve it
- ✅ Great for apps behind a login (dashboards, admin tools)
- ❌ Empty first paint — spinners while JS downloads and data fetches
- ❌ Poor SEO — crawlers see an empty shell
- ❌ Request waterfalls — render, fetch, render, fetch…
