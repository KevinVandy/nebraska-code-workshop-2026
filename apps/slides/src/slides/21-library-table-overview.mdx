- **Headless UI Library** — the table engine; you own 100% of markup and styles
  - Works with pretty much any Component Library or UI Framework
  - Bring your own design system
- Modular and Tree-Shakable: register only the features you use, the rest tree-shakes away
- Features like sorting, filtering, pagination, selection, expanding, grouping, pinning, resizing, aggregation…
- Works with built-in client-side features or your own manual server-side logic

```tsx
import { tableFeatures, useTable } from "@tanstack/react-table"

const features = tableFeatures({}) // opt-in!

function PersonTable() {
  const table = useTable({
    features,
    columns,
    data,
  })
  return table
    .getRowModel()
    .rows.map((row) =>
      row.getAllCells().map((cell) => <table.FlexRender cell={cell} />)
    )
}
```
