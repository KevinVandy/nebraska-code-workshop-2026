- **Headless UI Library** — the virtualizer engine; you own 100% of markup and styles still
- Renders only the visible rows — 10,000+ item lists, tiny DOM
- Vertical, horizontal, or both axes combined for grid virtualization
- Can handle dynamic sizes, item counts, and more

```tsx
import { useVirtualizer } from "@tanstack/react-virtual"

function List() {
  const parentRef = useRef(null)
  const virt = useVirtualizer({
    count: 10000,
    getScrollElement: () => parentRef.current,
    estimateSize: () => 35,
  })
  // inner div sized to virt.getTotalSize(),
  // render virt.getVirtualItems()
}
```
