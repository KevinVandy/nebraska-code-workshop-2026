<LibraryOverviewSlide library="virtual">

- Renders only the visible rows — 10,000+ item lists, tiny DOM
- Headless utility, not a component: zero markup or styles shipped
- Vertical, horizontal, or both axes combined for grid virtualization
- Dynamic sizing via `estimateSize` + measured items
- Recipe: scroll container → inner div at `getTotalSize()` → position `getVirtualItems()`

```tsx
import { useVirtualizer } from "@tanstack/react-virtual"

function List() {
  const parentRef = useRef(null)
  const virt = useVirtualizer({
    count: 10000,
    getScrollElement: () => parentRef.current,
    estimateSize: () => 35,
  })
  // inner div sized to virt.getTotalSize(),
  // render virt.getVirtualItems()
}
```
