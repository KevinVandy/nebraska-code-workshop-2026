## TanStack Product Tenets

- **Open, Independent, Technology-Agnostic**
  - All libraries are written in Vanilla JavaScript (TS)
  - Goal is to be agnostic to all other code in your codebase
  - You should be able to use TanStack libraries with any framework, CSS, Component library, design system, other 3rd party libraries, etc.
- **Composable, Platform-Aligned Primitives**
  - Your code using TanStack libraries should be easy to integrate into your codebase
  - TanStack libraries should have escape hatches and be easy to override or customize
- **Pragmatic, Production-Grade Quality**
  - We care about performance, scalability, security, and maintainability
- **Predictable, Explicit, Type-Safe Behavior**
  - Type Safety is a number one priority
  - The code should do what it sounds like it should do
