<LibraryOverviewSlide library="form">

- **Headless UI Library** — the form engine; you own 100% of markup and styles
- First-class TypeScript — field names inferred from `defaultValues`
- Sync + async per-field validation with built-in debouncing
- Schema validation with Zod, Valibot, ArkType, or any Standard Schema
- Fine-grained reactivity: `form.Subscribe` re-renders only what changed
- `createFormHook` pre-binds your own field components to kill boilerplate

```tsx
import { useForm } from '@tanstack/react-form'

const form = useForm({
  defaultValues: { firstName: '' },
  onSubmit: ({ value }) => console.log(value),
})

<form.Field name="firstName" children={(f) => (
  <input
    value={f.state.value}
    onChange={(e) =>
      f.handleChange(e.target.value)} />
)} />
```
