## Which Apps Should Be Built with Which?

- **CSR (SPAs)** — dashboards, admin tools, anything behind a login
- **SSG** — marketing sites, blogs, docs — content that changes on deploy
- **SSR** — public, dynamic, SEO-critical — e-commerce, listings, feeds
- **RSCs / Streaming** — content-heavy pages with islands of interactivity
- There is no one right answer — pick per app (or per _route_)
